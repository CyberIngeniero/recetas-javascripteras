# 🎡 For...in loop

for..in iterates over all enumerable property keys of an object

```js
const animals = ["🦁", "🦖", "🦈"];
const pet = {
  name: "Simba",
  emoji: "🦁",
};

for (let arrKey in animals) {
  console.log(arrKey); // 0, 1, 2
}

for (let objKey in pet) {
  console.log(objKey); // 'name', 'emoji'
}
```
