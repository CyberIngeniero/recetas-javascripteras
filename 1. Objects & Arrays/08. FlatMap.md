# 📦 Flat Array

Let's start with some tasks:

```js
const tasks = ['🧺', '🏋️', '🧑🏼‍🍳'];
```

## Before flatMap

```js
const doneTasks = tasks.map(task => [task, '✅']).flat();
console.log(doneTasks);
// ["🧺", "✅", "🏋️", "✅", "🧑🏼‍🍳", "✅"] 
```

## Now with flat map

```js
const doneTasks = tasks.flatMap(task => [task, '✅']).flat();
console.log(doneTasks);
// ["🧺", "✅", "🏋️", "✅", "🧑🏼‍🍳", "✅"] 
```
