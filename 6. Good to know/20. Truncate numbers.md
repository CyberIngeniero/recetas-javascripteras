# 🪓 Truncate number

Truncate a number is removing its decimal part.

## 💾 Bitwise version 

```js
console.log(15.32 | 0); // 15
```

❗️ Only for numbers up to 2^31.

## 🦸🏼‍♀️ Math object

```js
console.log(Math.trunc(15.32)); // 15
```

100% safe
