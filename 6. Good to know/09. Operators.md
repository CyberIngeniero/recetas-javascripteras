# ➕ Operators

## Increment and decrement

The order matters:

```
let a = 1;
console.log(++a, a); // 2 2

let b = 1;
console.log(b++, b); // 1 2
```

We could also use decrement operator:

```
let c = 1;
console.log(--c, c); // 0 0

let d = 1;
console.log(d--, d); // 1 0
```

## Addition and subtract assignment

```
let a = 1;
a += 2;
console.log(a); // 3

let b = 10;
b -= 4;
console.log(b); // 6
```

And we could also multiply, divide and calculate the module:

```
let a = 3;
a *= 2;
console.log(a); // 6

let b = 10;
b /= 2;
console.log(b); // 5

let c = 11;
c %= 10;
console.log(c); // 1
```
