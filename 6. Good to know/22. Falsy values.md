# 🦉 Falsy values. || and ?? operators

## || operator

```js
const usersCount = 0;
console.log(usersCount || 'Cargando usuarios');
// 😡 0 is a falsy values, so loading message is shown
```

## ?? operator

```js
const usersCount = 0;
console.log(usersCount ?? 'Cargando usuarios');
// 😁 0 is not null nor undefined
```
