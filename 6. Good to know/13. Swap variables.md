# 游대 Swap variables

## Old skool way

```js
var currentMood = '游뗴';
var mood = '游때';

var aux = currentMood;
currentMood = mood;
mood = aux;

console.log(currentMood); // '游때'
console.log(mood); // '游뗴'
```

## Cool way

```js
var currentMood = '游뗴';
var mood = '游때';

[currentMood, mood] = [mood, currentMood];

console.log(currentMood); // '游때'
console.log(mood); // '游뗴'
```
