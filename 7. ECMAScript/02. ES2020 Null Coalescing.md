# 🕳️ ES2020. Null coalescing operator

## 1️⃣ Undefined

```
var undefinedVar;
console.log(undefinedVar ?? "Default"); // "Default"
```

## 2️⃣ Null

```
const nullVar = null;
console.log(nullVar ?? "Default"); // Default
```

## 3️⃣ Empty String

```
const emptyString = '';
console.log(emptyString ?? "Default");  // "" ❗❗❗
```

## 4️⃣ Zero

```
const zero = 0;
console.log(zero ?? "Default"); // 0 ❗❗❗
```

## 5️⃣ Falsy value

```
const falsy = false;
console.log(falsy ?? "Default"); // false ❗❗❗
```
